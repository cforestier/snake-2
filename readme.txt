Snake2
Welcome to Snake2, an improved version of the classic Snake game.

How to Play
Use the arrow keys to steer the snake and eat the bonus and jokers to grow and make points. Avoid biting your tail!

Controls
Use the arrow keys to move
click the "start your 3310!" button to start playing
you can reload the game with the second button
